# アプリ名: student-score-management

概要:
教師向けのテストデータ管理サービス
最終的に統計情報を用いて各生徒の学習状況を把握できる｡
データを用いて効率的な生徒のサポートが見込める｡

1. データ登録アプリケーションにて登録するデータ

```
教科
- 教科名 subject
	- 国語､数学
- 担当教師 teacher
	- 山田ごんざぶろう

生徒
- 名前 name
	- 佐藤たろう
- 学年 grade
	- 1年､2年､3年

テスト
- 生徒番号 studentId
- 教科ID subjectId
- 種類
	- 一学期中間､二学期期末､学年末
- 点数 score
	- 0 ~ 100
```
